1. Platform Overview

LifeVibe is a modern blog platform designed for sharing content across three main lifestyle categories. The platform features a React-based frontend with a Node.js/Express backend, providing a smooth and responsive user experience.

Key Features:

  • Content Management: Create, edit, and delete blog posts
  • Category Organization: Posts organized into Travel, Food, and Wellness categories
  • Responsive Design: Mobile-friendly interface using Bootstrap
  • Admin Authentication: Secure login system for administrators
  • Image Support: Add images to posts via URL links
  • Real-time Updates: Immediate reflection of changes on the live site

2. Accessing the Admin Panel

2.1 Login Process

Step 1: Navigate to the admin login page by visiting: [website-url]/admin/login
Step 2: Enter your admin credentials:
  • Username: Your assigned admin username
  • Password: Your secure admin password
Step 3: Click the "Login" button to access the admin dashboard
Security Note: Your admin session will automatically expire for security purposes. You'll need to log in again if you've been inactive for an extended period.

2.2 Login Troubleshooting

Issue Solution
Invalid username/password error Double-check credentials and contact system administrator if needed
Page won't load Check internet connection and try refreshing the page
Redirected to homepage Clear browser cache and cookies, then try again

3. Dashboard Navigation

Once logged in, you'll have access to the main dashboard with the following navigation options:

Navigation Menu:

  • Home: View the public blog homepage
  • Add Post: Create new blog content
  • Manage Posts: View, edit, or delete existing posts
  • Logout: Securely exit the admin panel

4. Content Management

The LifeVibe platform allows you to manage blog posts across three predefined categories:

Category Description Content Examples
Travel Travel experiences, destinations, tips Destination guides, travel stories, itineraries
Food Culinary content, recipes, restaurant reviews Recipes, food reviews, cooking tips
Wellness Health, fitness, mental well-being content Workout routines, meditation guides, health tips

5. Creating New Posts

5.1 Accessing the Post Creation Form

Step 1: Click "Add Post" in the navigation menu
Step 2: You'll be redirected to the post creation form

5.2 Filling Out the Post Form

Required Fields:

Title:

  • Enter a compelling and descriptive title for your post
  • Keep titles concise but informative (recommended: 50-60 characters)
  • Use title case capitalization

Content:

  • Write your main blog content in the large text area
  • Use clear paragraphs and proper formatting
  • Include engaging introduction and conclusion
  • The system supports plain text formatting

Category:

  • Select from the dropdown: Travel, Food, or Wellness
  • Choose the category that best fits your content theme

Author:

  • Enter the author's name or pen name
  • Be consistent with author naming across posts

Image URL (Optional):

  • Provide a direct link to an image that represents your post
  • Ensure the image URL is publicly accessible
  • Recommended image dimensions: 800x600 pixels or similar aspect ratio
  • Supported formats: JPG, PNG, WEBP

5.3 Publishing Your Post

Step 1: Review all fields for accuracy and completeness
Step 2: Click the "Create Post" button
Step 3: Wait for the success confirmation message
Step 4: Your post will be immediately live on the website

6. Editing Existing Posts

6.1 Finding Posts to Edit

Step 1: Navigate to the homepage to view all published posts
Step 2: Click on any post title to view the full post
Step 3: Look for the "Edit" button (visible only to logged-in admins)

6.2 Making Edits

Step 1: Click the "Edit" button on the post you want to modify
Step 2: The edit form will load with current post data pre-filled
Step 3: Make your desired changes to any field
Step 4: Click "Update Post" to save changes
Note: All changes are saved immediately and will be reflected on the live website. There is no draft or preview mode, so ensure your edits are final before saving.

7. Deleting Posts

Warning: Post deletion is permanent and cannot be undone. Always double-check before deleting content.

7.1 Deletion Process

Step 1: Navigate to the post you want to delete
Step 2: Click the "Delete" button (visible only to logged-in admins)
Step 3: Confirm the deletion when prompted
Step 4: The post will be immediately removed from the website

8. Managing Categories

The LifeVibe platform currently supports three fixed categories: Travel, Food, and Wellness. These categories cannot be modified through the admin interface.

Category Guidelines:
  • Travel: Use for content about destinations, travel experiences, tips, and itineraries
  • Food: Use for recipes, restaurant reviews, cooking techniques, and culinary experiences
  • Wellness: Use for health tips, fitness routines, mental wellness, and lifestyle advice

9. Best Practices

9.1 Content Quality

  • Proofread: Always check for spelling and grammar errors before publishing
  • Engaging Titles: Create titles that are both informative and compelling
  • Consistent Voice: Maintain a consistent tone and style across all posts
  • Appropriate Length: Aim for 300-1500 words depending on the topic
  • Clear Structure: Use clear paragraphs and logical flow

9.2 Image Management

  • Quality Images: Use high-resolution, relevant images
  • Copyright Compliance: Ensure you have rights to use all images
  • Consistent Sizing: Try to use images with similar aspect ratios
  • Descriptive Images: Choose images that clearly relate to your content

9.3 SEO Considerations

  • Descriptive Titles: Include relevant keywords in your titles
  • Category Relevance: Ensure posts are in the most appropriate category
  • Content Depth: Provide comprehensive, valuable information
  • Regular Updates: Keep content fresh and up-to-date

10. Troubleshooting

10.1 Common Issues and Solutions

Problem Possible Cause Solution
Cannot access Add Post page Not logged in as admin Ensure you're logged in with admin credentials
Image not displaying Invalid or inaccessible image URL Check the image URL is correct and publicly accessible
Post creation fails Missing required fields Ensure all required fields (title, content, category, author) are filled
Cannot see Edit/Delete buttons Not logged in as admin Log in with admin credentials
Changes not appearing Browser cache Clear browser cache or hard refresh the page

10.2 Browser Compatibility

The LifeVibe admin panel is optimized for modern browsers. For best experience, use:

  • Chrome (version 90+)
  • Firefox (version 88+)
  • Safari (version 14+)
  • Edge (version 90+)

11. Support Information

11.1 Getting Help

If you encounter issues not covered in this guide:

Step 1: Check this user guide for relevant solutions
Step 2: Try basic troubleshooting (refresh page, clear cache)
Step 3: Contact the system administrator with:
  • Detailed description of the issue
  • Steps you took before the problem occurred
  • Screenshots if applicable
  • Browser and operating system information

11.2 Security Reminders

  • Keep Credentials Secure: Never share your admin login details
  • Log Out When Done: Always log out when finished with admin tasks
  • Use Strong Passwords: Ensure your admin password is strong and unique
  • Report Suspicious Activity: Contact support if you notice unusual activity

11.3 System Maintenance

The platform may occasionally undergo maintenance. During these periods:

  • Admin functions may be temporarily unavailable
  • You will receive advance notice when possible
  • The public website will remain accessible for viewing
  • Any unsaved work should be saved locally as a backup
